# Objective

No roles or permissions system exists. Add role-based access control (RBAC): define roles (`admin`, `member`), permission sets, and FastAPI dependencies for role checks. Integrate with Clerk user metadata or a local user table `role` field. The system should be extensible for future roles without code changes to the dependency layer.

# Files

Source files:
- `backend/app/core/rbac.py` (new) -- role definitions, permission sets, and FastAPI dependencies
- `backend/app/users/models.py` -- add `role` field to user model (default `member`)
- `backend/app/core/auth.py` -- extend with `require_role()` dependency that checks role after auth

Test files:
- `backend/tests/core/test_rbac.py` (new) -- tests for role definitions and permission checks
- `backend/tests/core/test_auth_roles.py` (new) -- tests for FastAPI role-check dependencies

# Implementation Plan

1. Write tests for role and permission definitions:
   - `Role` enum has `ADMIN` and `MEMBER` values
   - `ROLE_PERMISSIONS` maps each role to a set of permission strings
   - `ADMIN` role has all permissions that `MEMBER` has, plus admin-specific ones
   - `has_permission(role, permission)` returns `True`/`False` correctly
2. Implement `rbac.py` with `Role` enum, permission sets, and `has_permission()` helper
3. Write tests for the user model role field:
   - User model has a `role` field defaulting to `"member"`
   - Role field accepts `"admin"` and `"member"` values
4. Update `users/models.py` to add the `role` column
5. Write tests for `require_role()` FastAPI dependency:
   - `require_role(Role.ADMIN)` returns a dependency callable
   - Dependency raises `HTTPException(403)` when user role is `MEMBER` and `ADMIN` is required
   - Dependency passes through when user role matches required role
   - Dependency works with the existing auth dependency chain
6. Implement `require_role()` in `auth.py` or `rbac.py` as a FastAPI `Depends` factory
7. Write tests for `require_permission()` dependency:
   - Checks if the user's role has the required permission
   - Raises 403 if permission not in role's permission set
