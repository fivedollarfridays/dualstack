# Implementation Plan

1. Write `tests/test_dist.py` with failing tests:
   - `test_dist_script_exists_and_executable`
   - `test_dist_excludes_internal_files` — parse script for exclusion patterns
   - `test_dist_includes_required_files` — parse script for expected content
   - `test_makefile_has_dist_target`
2. Create `scripts/dist.sh`:
   - Accepts optional version arg, defaults to date
   - Creates `dist/dualstack-<version>.zip`
   - Excludes: `.paircoder/`, `.claude/`, `CLAUDE.md`, `AGENTS.md`, `.git/`, `node_modules/`, `.venv/`, `__pycache__/`, `.env*` (non-example), `*.db`, monitoring data, planning docs
   - Includes: `backend/`, `frontend/`, `monitoring/`, `scripts/`, `docs/`, `badges/`, `Makefile`, `package.json`, `README.md`, `GETTING_STARTED.md`, `LICENSE`, `SECURITY.md`
3. `chmod +x scripts/dist.sh`
4. Add `dist` target to Makefile
5. Confirm `dist/` in .gitignore
6. Run tests
