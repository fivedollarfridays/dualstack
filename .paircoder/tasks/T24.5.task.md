Remove npm lockfile, generate pnpm lockfile. Makefile and docs use pnpm but frontend has package-lock.json — standardize on pnpm.

# Files to Update

| File | Change |
|------|--------|
| `frontend/package-lock.json` | Delete |
| `pnpm-lock.yaml` | Generated by `pnpm install` |
| `tests/test_buyer_experience.py` | Add lockfile tests |

# Implementation Plan

1. Add failing tests to `tests/test_buyer_experience.py`:
   - `test_no_npm_lockfile` — assert `frontend/package-lock.json` does not exist
   - `test_pnpm_lockfile_exists` — assert pnpm-lock.yaml exists at root
2. Delete `frontend/package-lock.json`
3. Run `pnpm install` from project root to generate `pnpm-lock.yaml`
4. Verify frontend tests still pass: `pnpm test`
5. Run buyer experience tests
