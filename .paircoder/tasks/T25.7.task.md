# Objective

Add React error boundaries for graceful degradation and fix display_name to store raw Unicode instead of HTML-escaped values.

# Implementation Plan

1. Create `ErrorBoundary` component with user-friendly fallback UI and retry button
2. Wrap dashboard layout with error boundary
3. In `users/schemas.py`, remove `html.escape()` from `display_name` validator — strip only `<` and `>` characters (XSS prevention without entity encoding)
4. Update any tests asserting HTML-escaped display names
