# Objective

Improve accessibility with focus trap in ConfirmDialog and optimize avatar images with next/image.

# Implementation Plan

1. Add focus trap to `ConfirmDialog` — track focusable elements, cycle Tab within dialog
2. Replace `<img>` with `next/image` in `AvatarDisplay`
3. Configure `remotePatterns` in `next.config.mjs` for avatar domains (Clerk `img.clerk.com`, Gravatar `gravatar.com`)
4. Add proper `width`/`height` and `loading="lazy"` attributes
