# Objective

Replace the static `DEV_TOKEN = 'dev-token'` with a randomly generated per-session token.

# Implementation Plan

1. In `frontend/src/lib/auth-config.ts`, replace `const DEV_TOKEN = 'dev-token'` with `crypto.randomUUID()` generated once at module load
2. Keep `getDevToken()` and `isLocalDev()` guards unchanged
3. Update any tests that assert the exact dev token value

# Acceptance Criteria

- [ ] Dev token is randomly generated, not a static string
- [ ] Token is stable within a browser session (generated at module load)
- [ ] `getDevToken()` still returns null on non-localhost
- [ ] Tests updated to not assert exact token value
- [ ] All frontend tests pass
