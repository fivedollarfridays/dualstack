# Objective

Create `ClerkAuthBridge` (bridges Clerk `useAuth` into `AuthContext`), `DevAuthProvider` (mock auth: userId `dev-user-001`, getToken returns `dev-token`), `DevUserButton` (simple "D" avatar), `DevSignIn` (dev mode card with "Continue to Dashboard" button), `DevSignUp` (same pattern), `AppUserButton` (conditional: `UserButton` or `DevUserButton`). TDD -- tests first.

- `dev-auth-provider.tsx` -- mock auth provider for dev mode

- `dev-sign-in.tsx` -- dev mode sign-in card with Continue to Dashboard
