# Objective

Replace all `import { useAuth } from '@clerk/nextjs'` with `import { useAppAuth } from '@/contexts/auth-context'` in consumer files. Update sign-in/sign-up pages to conditionally render Clerk or dev stubs. Update dashboard layout to use `AppUserButton`. Update all corresponding tests to mock `@/contexts/auth-context` instead of `@clerk/nextjs`.
