# Objective

Fix three deployment issues: missing .dockerignore ships tests/caches into Docker image, test deps are in production requirements, and fastapi-clerk-auth is unpinned. Covers issues H9, M16, M11.

# Implementation Plan

## 1. Create backend/.dockerignore (H9)

Create `backend/.dockerignore` excluding:
```
__pycache__
*.pyc
.pytest_cache
.coverage
.coveragerc
pytest.ini
tests/
.env*
htmlcov/
*.egg-info
```

## 2. Split requirements (M16)

Create `backend/requirements-dev.txt`:
```
-r requirements.txt
pytest>=8.0
pytest-asyncio>=1.0
pytest-cov>=7.0
pytest-timeout>=2.0
```

Remove test deps from `backend/requirements.txt`. Update Dockerfile to only install `requirements.txt`. Update README test instructions to mention installing dev deps.

## 3. Pin fastapi-clerk-auth (M11)

Change `fastapi-clerk-auth>=0.0.9` to `fastapi-clerk-auth==0.0.9` in requirements.txt.
