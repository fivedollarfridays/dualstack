# Objective

Close two High-severity findings: disable OpenAPI/Swagger UI in production (NEW-002) and document the X-Forwarded-For trust model with a restricted `--forwarded-allow-ips` comment (NEW-001).

# Implementation Plan

1. In `main.py`, set `docs_url=None`, `redoc_url=None`, `openapi_url=None` when `settings.environment == "production"`
2. In `entrypoint.sh`, add comment explaining that `--forwarded-allow-ips='*'` relies on Render's load balancer overwriting X-Forwarded-For and that direct-to-container access must be blocked at the network level
3. Verify `rate_limit.py` docstring already documents proxy trust (done in prior review pass)
4. Write tests: production mode returns 404 for `/docs`, `/redoc`, `/openapi.json`; dev mode returns 200
